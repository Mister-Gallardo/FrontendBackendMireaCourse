# ЗдоровоЕда — Сервис доставки здорового питания

## О проекте

Многостраничный адаптивный веб-сайт для сервиса доставки здорового питания, разработанный в рамках учебной практики.

## Используемые технологии

- **CSS-фреймворк:** Bootstrap 5.3.2
- **Иконки:** Bootstrap Icons 1.11.1
- **Подключение:** CDN

## Структура проекта

```
FrontendBackendMireaCourse/
├── index.html        # Главная страница (лендинг)
├── contacts.html     # Страница контактов с формой обратной связи
├── faq.html          # Страница FAQ с аккордеоном
├── 404.html          # Страница ошибки 404
├── assets/
│   └── css/
│       └── styles.css  # Кастомные стили
│   └── icons/
│       └── food-icon.svg  # Иконка
└── README.md
```

## Страницы сайта

### Главная (index.html)

- Фиксированный header с навигацией
- Hero-секция с CTA-кнопкой
- Секция «О сервисе»
- Секция преимуществ (4 карточки)
- Footer с контактами

### Контакты (contacts.html)

- Контактная информация (адрес, телефон, email)
- Ссылки на соцсети
- Форма обратной связи с валидацией

### FAQ (faq.html)

- Аккордеон Bootstrap с 6 вопросами и ответами
- Кнопка перехода на страницу контактов

### 404 (404.html)

- Стилизованная страница ошибки
- Кнопка возврата на главную
- Ссылки на основные страницы
